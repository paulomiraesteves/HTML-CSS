  print(f'O número {num} é ÍMPAR')

Nesse caso, foi um texto em PYTHON. Se eu só usasse a tag CODE, o texto ia ficar igual um texto normal. Agora, já que envelopei a tag CODE com a tag PRE, o texto ficou assim: uma linha em cada espaço!
